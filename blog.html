What is the Document Object Model (DOM)?

Document Object Model is a programming interface for web documents, DOM representing the structure of a webpage as a tree of objects.The DOM is essential for building interactive web applications, enabling developers to create responsive and dynamic user experiences

How do you select an element from the DOM?

I can select element from DOM in various aways. the aways are :

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is a technique in the DOM where a single event listener is added to a parent element.When an event occurs on a child element, it bubbles up to the parent, allowing the parent to handle the event.

    Benefit of Event Delegation :

  • Reduces the number of event listeners
  • improving performance, especially with many child elements.
  • Simplifies handling of dynamically added elements
  • Reduces clutter in our code.

    Manipulating an element's attributes and styles using the DOM is easy with JavaScript.

  • I can use methods like setAttribute(), getAttribute(), and removeAttribute() to manipulate an element's attributes.
  • i can manipulate styles using the style property of an element.
  • adding or removing CSS classes with classList.
